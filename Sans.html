0xA000

Pixelated Sans Mono MonoSans

VGA compatible graphics hardware have mode 13h: 320x200 pixels with 256 colors, you pushed pixels onto the display by writing into memory segment 0xA000. When pixeling fonts for use there, I imagined shapes not dissimilar from what this font now contains.
