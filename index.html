0xA000

Pixelated Sans Mono MonoSans

Hi-fidelity low-PPI simulator

The shapes for the glyphs in this font have been designed within the constraints of legibility when lowercase is rendered 3 pixels high.

  • Can be used for typesetting tiny compact text
  • Provides a retro blocky pixel text
  • Compact files, each TTF is smaller than 64kb containing ASCII + Latin1 converage ++
  • TTFs of all the various fonts used in this document are linked at the bottom.
  • NB this page is editable; so you can experiment typing with the typeface yourself.
  • This list use the • (Bullet) character from the font.
  • The font's are designed to be legible without any hinting required at sizes which are a multiple of 5px. With hinting; things might work better or worse for sizes in between.
